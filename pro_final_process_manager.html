고객용 APP

소개

특장차 제조 업체에서 계약건별 공정 현황을 쉽게 확인할 수 있는 애플리케이션입니다.
지금까지 고객은 차량 제작이 어디까지 진행됐는지 확인하고 싶으면 매번 업체에 전화를 걸어 확인해야 했습니다. 업체 측에서는 작업 현장에 연락하여 알게 된 진행 상황을 고객에게 다시 전달해야 하는 불편함이 있었습니다.
이 APP을 사용하면 작업을 완료한 뒤 버튼 하나만 누르면 작업 내역이 등록됩니다. 관리자가 작업 내역을 승인하면 다음 공정 작업자에게 자동으로 업무가 인수되며, 고객은 PUSH 알림으로 진행 내역을 전달받을 수 있습니다.
직원을 위한 업무용 APP과 고객 확인용 APP을 나누어 제작했습니다.
제작기간 : 2023.04.20 - 06.02

팀원 : 강유준, 최병권, 이소우, 강성중, 김소윤

담당 파트
(API) 공정 관리, 패키지 관리, PUSH 알림 연동, 로그인
(APP) 고객용 앱 로그인, 진행상황 확인 목록

사용 기술

Back-end : Java https://gitlab.com/all_xm/finalprojectapi
Front-end(APP) : Flutter https://gitlab.com/all_xm/finalprojectappcustomer
API : Firebase cloud messaging
